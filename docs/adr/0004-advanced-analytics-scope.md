# ADR 0004: Advanced Analytics Included in MVP

## Status
Accepted

## Context
The project requires not only basic returns but deeper analytics from early iterations.

## Decision
Include advanced analytics in MVP:

- TWR (time-weighted return)
- MWR/IRR (money-weighted return)
- benchmark spread
- drawdown
- rolling volatility
- concentration indicators

## Consequences

- Positive: analytics value is available from first usable version.
- Negative: larger validation and test scope for financial correctness.
